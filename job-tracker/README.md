# Blake's Job Tracker

A self-contained job application tracker for finding and tracking data / cloud / AI / consulting
roles in **Houston, Austin, and Dallas, TX** — targeting Fortune 500 companies, PwC and other
consulting firms, and Texas startups.

## How to use it

**Option 1 — open locally:** download `index.html` and double-click it. Everything runs in your
browser; no install, no server, no account.

**Option 2 — GitHub Pages:** enable Pages for this repo (Settings → Pages → deploy from `main`)
and visit `https://worldwideblake.github.io/Worldwideblake/job-tracker/`.

## Features

- **Pre-seeded pipeline** — companies and roles verified against the company's own careers site
  (seeded July 14, 2026). Every row has an **Apply ↗** link to the posting and a **careers ↗**
  link to the company's job-search page pre-filtered where possible.
- **Status tracking** — Not Applied → Saved → Applied → Interviewing → Offer / Rejected / Closed.
  Setting a row to *Applied* auto-stamps today's date.
- **Stat tiles** — click a tile (Applied, Interviewing, …) to filter to those rows.
- **Search & filters** — by city, category (Fortune 500 / Consulting / Startup), and status.
- **Add your own jobs** with the *+ Add job* button.
- **Notes and applied-on date** editable inline in the table.
- **Export CSV** for spreadsheets; **Backup/Restore JSON** to move your data between browsers
  or devices (data lives in the browser's localStorage).
- Light & dark mode.

## Keeping it accurate

Job postings close quickly. Before applying, always click through and confirm the role is still
open on the company's site — the **careers ↗** link on each row takes you to the company's live
search so you can also spot newly opened roles. When a posting closes, set its status to
**Closed** and check the careers link for its replacement.
